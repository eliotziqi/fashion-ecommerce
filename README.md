# 女装电商网站

一个面向中国消费者的女装电商网站，参考 Zara 和 H&M 的设计风格。

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React (图标库)

## 功能特性

- 🏠 首页：轮播图、新品推荐、热销商品、新用户弹窗
- 👤 用户系统：邮箱、微信、支付宝注册登录
- 🛍️ 商品浏览：商品列表、商品详情（图片、尺码、评价）
- 🛒 购物流程：购物车、下单结算、支付（支付宝、微信、银行卡）
- 📦 订单管理：收货地址、历史订单
- 📧 联系支持：邮件联系官方

## 设计风格

- 主色调：黑白灰
- 点缀色：米色、深蓝色
- 大图展示商品
- 简洁布局
- 流畅交互动效
- 统一 icon 风格

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

访问 [http://localhost:3000](http://localhost:3000) 查看网站。
